# 中移物联网 OpenCPU SDK 简介

## **注意事项（必读）**

### 当前支持编译型号为SDK开发包名称中的子型号

- 编译命令：scons 
- 编译示例程序：scons demo=xxx。xxx为示例名称，例 scons demo=breathled，编译呼吸灯示例
- 编译测试程序：scons test=y。
- 清除编译：scons demo=xxx -c清除编译示例，或scons test=y -c清除编译测试程序
- 内核编译命令：kernel_build.bat ML307C
- 清除编译：kernel_build.bat clean

### GNSS功能说明

ML307C GNSS芯片的固件会在开机时执行动态加载。对应的GNSS固件和boot文件内置在模组的文件系统之中，请勿删除或移动该文件，若修改会导致GNSS功能失效。对应两个文件如下：
1、\\.cm_etc\gnss\cc1165w_Firmware.pkg
2、\\.cm_etc\gnss\CC11_Bootloader.pkg

### 外置codec芯片使用说明

1. SDK支持PWM接口输出音频模拟信号（PWM输出模拟音频信号模式）驱动扬声器方案（默认）和使用外挂音频设备（例如codec芯片）实现音频能力（输出PCM数字音频信号模式）方案；
2. 外挂音频设备提供音频能力方案为PCM接口输出音频数字信号至用户外挂设备由用户外挂音频设备执行播放；
3. 外挂音频设备提供音频能力方案下，模组自身不具备音频能力，仅通过PCM接口与外挂音频设备进行PCM数据的通信；
4. 外挂音频设备提供音频能力方案下，音频播放由外挂音频设备完成，相关音效、降噪、校准等能力请咨询外挂音频设备厂商；
5. 用户外挂音频设备需要驱动才可使用，可使用AudioHAL_AifBindCodec_CB()注册用户外挂音频设备开启/关闭操作的回调函数，底层在相应操作下自行调用用户注册的回调；
6. cm_ex_codec.c中默认提供“ES8311 CODEC芯片+无降噪芯片”方案，用户使用此方案仅需完成IIC从设备地址和ES8311 CODEC参数适配后即可使用；
7. 外挂音频设备提供音频能力方案下，设备上电需调用vpathSetMode(1)接口进行设置，可参考int vpathSetMode(uint32_t mode)接口注释和demo使用；
8. 外挂音频设备（例如codec芯片）实现音频能力方案下，仅支持音频播放，不支持录音功能。
