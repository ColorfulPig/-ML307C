
Activate PPC: (same as window version)
1. PPC.exe <input file> <output file>
2. PPC.exe <@input list file>
3. PPC.exe <@input list file> <output path>

Activate PPL:
1. PPL.exe <TxtDB.txt file path> <Output files path> [@<input file name>] [pp file1] ... [pp files] [-c:<first command ID>] [-r:<first report ID>]
2. PPL.exe <TxtDB.txt file path> <Output files path> [@<input file name>] [pp file1] ... [pp files] [-f for filter file] [filter file1] [-f] [filter file2] [-c:<first command ID>] [-r:<first report ID>]

the first parameter <TxtDB.txt> come in place of the first two parameters, DB names in window


Activate PPDBMerga:
PPDBMerge.exe <input1.txt> <input2.txt> .. <inputn.txt>

all the files are merge to the first file

the Input?.txt created by the PPL
