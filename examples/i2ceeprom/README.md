# 简易i2c操作eeprom(is24c256)简介 

## **模组i2c功能介绍**
本功能模块通过i2c接口驱动eeprom(is24c256)，主要目的是展示模组作为主设备i2c功能模块接口的使用方法;
本模组的i2c只能作为主设备，不能作为从设备，两线制（CLK+SDA),支持特定波特率和特定地址位数，详见cm_i2c.h；
本模组的i2c为标准协议，这里不再赘述。

## **实现功能**
实现模组通过i2c与eeprom的通讯，包括以下子功能：
1. 打开外设eeprom，包括引脚功能设置，i2c的参数设置等；
2. 修改外设eeprom特定地址上的数据；
3. 读取外设eeprom特定地址上的数据；
4. 关闭外设eeprom。

## **APP执行流程**
1. 设备上电，创建i2c操作eeprom任务，该任务间隔特定时间操作eeprom 1次；
2. 打开外设eeprom；
3. 修改外设eeprom特定地址上的数据；
4. 读取外设eeprom特定地址上的数据；
5. 关闭外设eeprom。

## **使用说明**
- 支持的模组（子）型号：ML307R各子型号/ML307C-DC-CN；
- 支持的SDK版本：ML307R OpenCPU SDK 2.0.0/ML307C OpenCPU SDK 1.0.0版本及其后续版本；
- 是否需要外设支撑：必须接eeprom(is24c256)，如果是其他外设，请阅读相关外设芯片手册调整相关配置参数；
- 使用注意事项：1.确保硬件连接无误；2.确保外设芯片与功能模块中的一致；
- APP使用前提：模组开发板及其外设eeprom(is24c256)芯片；

## **FAQ**
- 问题：除了关注cm_i2c.h中的接口函数，还需要注意什么？
  答复：1.需调用cm_iomux.h文件引脚配置接口设置选择引脚为i2c功能；
        2.需确保模组和芯片供电稳定和引脚连接正确，可用示波器抓取波形进行相关参数分析；
